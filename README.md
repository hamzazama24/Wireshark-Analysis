# Wireshark-Analysis
Analyzed 500+ DNS, HTTP, and TCP packets using Wireshark to identify anomalies and recommend security improvements.
Performed network traffic capture using Wireshark on a Wi-Fi interface.
Filtered DNS traffic to analyze domain name resolution activity.

Inspected individual packets and analyzed protocol layers including Ethernet, IP, UDP, and DNS.
Observed DNS responses for domains such as zorvian.com and cloudflare services, confirming normal web traffic behavior.

Reviewed packet-level details including source/destination IP addresses and query responses to understand how domain resolution works.
This project demonstrates hands-on experience in packet analysis, protocol inspection, and network traffic monitoring.
